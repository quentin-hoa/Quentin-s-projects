#  My Hunter - Graphical Game

## Description
My Hunter is a "Duck Hunt" style game. It is the first graphical project at Epitech, focusing on window management, sprites, and event handling using the CSFML library.

## Key Technical Skills
* **Event Handling:** Capturing mouse clicks and window closing events.
* **Sprite Animation:** Using "rects" to animate a sprite sheet (moving xwings).
* **Game Loop:** Implementing a consistent loop for rendering, events, and updates.



##  Constraints & Rules
* **Interaction:** The player must click on all moving targets to pass to the next level.
* **Progression:** The speed of the targets increases as the levels goes up.
* **Feedback:** The window must react to clicks (level update) and manage a "game over" state if too many targets are missed.

## How to Use

### 1. Compilation
Use the provided Makefile to compile the project:
* In a terminal: `make`

### 2. Execution
Launch the game:
* In a terminal: `./my_hunter`
* Use your mouse to click on the targets and use the `-h` flag to see available options.
